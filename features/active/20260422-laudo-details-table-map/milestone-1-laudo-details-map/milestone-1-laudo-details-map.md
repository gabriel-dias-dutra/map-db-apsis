# Milestone 1 — LaudoDetails Map (MVP)

## Description

The whole feature. Delivers the introspection script, the generated Portuguese map, and the n8n agent integration so the database search agent can answer laudos questions correctly from day one. A Softo developer, starting from an empty `repos/sql/laudo-details/` folder, runs two commands and then replaces a marker-delimited section in the n8n agent's system prompt — and the agent now knows how to reason about `[projects].[LaudoDetails]`, including the text-stored numeric and date columns that would otherwise produce silently wrong SQL.

**Deliverables included:**
- Deliverable 0001 — Schema introspection of LaudoDetails
- Deliverable 0002 — LaudoDetails map generation
- Deliverable 0003 — Agent prompt integration and operator README

**Complexity:** 3 deliverables — 1 Very Low · 2 Medium

**Size:** 3 deliverables — 1 Very Small · 2 Small
