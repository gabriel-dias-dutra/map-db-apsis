# Milestone 1 — Database Map (MVP)

## Description

Delivers the core knowledge layer: a generated, human-reviewable Markdown map of the Apsis SQL Server (~202 tables) that can be pasted into the n8n database search agent's system prompt, plus a validation battery to measure whether the agent actually answers business questions correctly. Execution is manual and on-demand by a Softo developer, reusing the existing Lambda SQL proxy as transport. No new infrastructure is introduced.

**Deliverables included:**
- Deliverable 0001 — Schema extraction
- Deliverable 0002 — Database Map generation
- Deliverable 0003 — Validation question battery
- Deliverable 0004 — Operator guide

**Complexity:** 4 deliverables — 1 Very Low · 1 Low · 2 Medium

**Size:** 4 deliverables — 2 Very Small · 1 Small · 1 Medium
